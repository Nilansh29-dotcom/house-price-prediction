# House-price-prediction
Machine Learning model to predict house prices
 House Price Prediction

# Project Overview

This project uses Machine Learning to predict house prices based on features like area, location, number of rooms, etc.

##  Technologies Used

* Python
* Pandas
*  NumPy
* Scikit-learn
  

##  Steps Performed

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Building (Linear Regression)
* Model Evaluation

##  Model Performance

* R² Score: 0.65 
* Mean Absolute Error: Low
* Mean Squared Error: Low


##  How to Run

1. Install dependencies: pip install pandas numpy scikit-learn 

2. Run the notebook:
   HousingPriceLinearRegression.ipynb
